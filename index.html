HTML son las siglas de HipertText Makup Lenguage que puede traducirse como lenguaje de marcas o marcado de hipertexto.
El lenguaje HTML se emplea para crear las paginas web. Es muy facil ver el codigo de HTML de una pagina web, la opcion exacta cambia de un navegador a otro y tambien puede cambiar de una version a otra de un minimo navegador pero suelen tener un nombre simila.
HTML se comopone de etiquetas que se escriben entre simbolos menor que y mayor que.

Versiones de HTML

Tim Berners-Lee definio la primera version de HTML en el año 1991 y en la actualidad la ultima version del HTML es HTML5
